# jQuery
## 개념
모든 브라우저에서 동일한 동작을 수행하게 하는 Client Side Javascript *Library*

> 플랫폼 vs 라이브러리 vs 프레임워크
> - 플랫폼: 다른 프로그램을 실행시켜줄 수 있는 Program
> - 라이브러리: 특정 기능을 실행하는 코드 조각
>   - 클래스 또는 함수
>   - 필요할 때 마다 가져와서 사용
> - 프레임워크: 시스템이 돌아가는 방식이 이미 정해져있고, 사용자는 그 규칙에 맞춰서 추가 코드를 작성해주어야 함
>   - 여러 개의 라이브러리를 포함

## 사용 목적
1. 화면 제어를 쉽게 => DOM을 동적으로 처리하는 것을 쉽게 할 수 있음
2. 이벤트 처리를 쉽게
3. AJAX(자바스크립트의 비동기 통신) 쉽게

모두 순수 자바스크립트로도 할 수 있으나, jQuery를 사용하면 보다 쉽게, 브라우저에 의존되지 않는 방식으로 구현 가능하다.

## 셀렉터
1. 전체 선택자 '*'
2. 타입 선택자 / '태그 이름'
3. ID 선택자 / '#ID'
4. 클래스 선택자 / '.CLASS'
5. 구조 선택자 (자식선택자와 후손선택자)
6. 동위 선택자 / +, ~
7. 동위 선택자 / +, ~
8. 속성 선택자 / []
